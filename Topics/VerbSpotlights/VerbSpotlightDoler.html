Verb Spotlight: Doler (Present Tense)

In this Verb Spotlight, we will learn about doler in the present tense.

Doler means “to hurt” or “to ache.” It typically uses indirect object pronouns to express who experiences the pain:

  • me duele / te duele / le duele / nos duele / os duele / les duele
It is also an o → ue stem changer, but you’ll mostly see it in the third-person forms (duele / duelen) when paired with pronouns.
