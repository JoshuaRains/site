Nouns in Spanish have genders? sort of.
In Spanish we put nouns in 2 groups that we call and .

Most other languages sort nouns into categories in this same way. In German, for example, nouns are sorted into the categories of "masculine," "neuter," and "feminine" For German the gender of most nouns simply has to be memorized individually.

We need to learn the grammatical genders of our nouns so that we can use them properly in sentences. Confusing the genders of the nouns could lead to confusion
Luckily for us, Spanish noun genders are usually pretty easy to spot. Usually if a noun ends in an it is masculine and if it ends in an it is feminine. If you memorize this rule, you will be able to figure out the gender of most nouns. There are exceptions that will will talk about later on.

Nouns in Spanish also have number: singular or plural. This should be more familar to you as an English speaker. In English we have to change the ending of our nouns when we want to make them plural:

car changes to cars
cup changes to cups
door changes to

In each of these examples we add an -s to make the noun plural. In spanish we do the same thing.

carro changes to carros
copa changes to copas
puerta changes to puertas

Did you notice the -o and -a on the end of the sigular nouns?
